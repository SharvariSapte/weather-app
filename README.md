# Weather App 

A javascript weather application that shows the current weather of any city using a weather API.

# Features

- Get current weather data for your location or any city that you enter
- Shows temperature, humidity, feels like, date and time
- message based on condition
- Uses geolocation to detect current location

# Technologies used

- HTML
- CSS
- Vanilla JavaScript
- Weather API
- Geolocation API
